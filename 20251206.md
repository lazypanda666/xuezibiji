# git基础篇

### 1. 提交与修改

| 命令                                  | 说明                                   |
| ------------------------------------- | -------------------------------------- |
| git commit                            | 提交暂存区到本地仓库                   |
| git branch <branchname>               | 建立分支                               |
| git branch                            | 查看分支                               |
| git checkout                          | 分支切换                               |
| git checkout -b <branchname>          | 建立并切换到该分支                     |
| git branch -d <branchname>            | 删除本地分支                           |
| git branch -D <branchname>            | 强制删除未合并的分支                   |
| git push origin --delete <branchname> | 删除远程分支                           |
| git merge <branchname>                | 合并分支                               |
| git rebase <branchname>               | 合并分支                               |
| git switch                            | 更清晰地切换分支                       |
| git add                               | 添加文件到暂存区                       |
| git reset                             | 回退版本                               |
| git rm                                | 将文件从暂存区和工作区中删除           |
| git mv                                | 移动或重命名工作区文件                 |
| git notes                             | 添加注释                               |
| git restore                           | 恢复或撤销文件的更改                   |
| git show                              | 显示Git对象的详细信息                  |
| git status                            | 查看仓库当前的状态，显示有变更的文件   |
| git diff                              | 比较文件的不同，即暂存区和工作区的差异 |
| git difftool                          | 使用外部差异工具查看和比较文件的更改   |
| git range-diff                        | 比较两个提交范围之间的差异             |

### 2. 提交日志

| 命令             | 说明                                                         |
| ---------------- | ------------------------------------------------------------ |
| git log          | 查看历史提交记录                                             |
| git blame <file> | 以列表形式查看指定文件的历史提交记录                         |
| git shortlog     | 生成简洁的提交日志摘要                                       |
| git describe     | 生成一个可读的字符串，该字符串基于Git的标签系统来描述当前的提交 |

### 3. 远程操作

| 命令          | 说明                      |
| ------------- | ------------------------- |
| git remote    | 远程仓库操作              |
| git fetch     | 从远程获取代码库          |
| git pull      | 下载远程代码并合并        |
| git push      | 上传远程代码并和并        |
| git submodule | 管理包含其他Git仓库的项目 |
